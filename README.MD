# botapp

Proyecto Java Maven para un bot de Telegram.

## Estructura

- `pom.xml` - configuración del proyecto Maven
- `src/main/java` - código fuente principal
- `src/test/java` - pruebas unitarias

## Ejecutar

```bash
mvn clean compile
mvn exec:java -Dexec.mainClass="com.zutjmx.App"
```

> Ajusta el comando de ejecución según tu entorno y configuración de Maven.
